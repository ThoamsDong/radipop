Uses three threshold values to find organs. 

The algorithm is\+:
\begin{DoxyItemize}
\item After some smoothing, remove every pixel above bones threshold from the image.
\item After some smoothing, remove every pixel above blood vessel threshold.
\item Everything that then remains above liver threshold is called an organ.
\item Use contiguous area divisions to roughly split into organs.
\end{DoxyItemize}
